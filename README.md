# Crmp-Mobile
Карта для игры Gta: San Andreas на смартфоне под названием Crmp Mobile!


Этот проект совершенно ни как не связан с другими проектами.

Я взял за основу карту которую перенесли из игры Gta: Criminal Russia на пк, и начал её доделовать.

Почему возникла такая идея?

Хочется создать почти свою игру чтобы в нее играли люди, причём все будет бесплатно.

Когда будет готов проект? 

К концу 2026 года
